Azamino Bakery – 統一UI一式（Hero含む）
====================================

同梱物
------
- index.html … 統一デザインのサンプル（ヘッダー／ヒーロー／メニュー／お知らせ／カレンダー／アクセス／問い合わせ／右サイド）
- css/style.css … すべての共通クラス（`ab-`プレフィックス）
- 画像は同梱していません。ヒーローは `images/hiro1-1280.jpg` を想定しています。無ければ placehold.jp が表示されます。

使い方
------
1. リポジトリに `index.html` と `css/style.css` を配置します。
2. ヒーロー画像を `images/hiro1-1280.jpg` に置くか、index.html のヒーローセクションの inline style（--hero-image）を書き換えてください。
3. PC時の右カラムは `.ab-layout` のCSS Gridで自動。スマホは1カラムです。
4. 既存ページへ導入する場合は、`<main>` 内の各セクションをコピーし、クラス名（ab-系）を付与していけば外観が統一されます。

調整ポイント
------------
- 最大幅：`:root` の `--max-w` を変更すると、ヒーローとコンテンツ幅が同時に揃います。
- ヒーロー高さ：`--hero-h-desktop`（デスクトップ）、`--hero-h`（モバイル）で調整。
- 色味：`--accent`, `--main-color`, `--highlight` を適宜。

備考
----
- Googleフォームやマップの埋め込みURLはダミーです。実URLに差し替えてください。
- スライダー化（ドット・矢印）は別途JS追加で対応可能です。
